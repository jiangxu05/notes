## 定义方式：
    1.数据类型 数组名[数组长度]；
    2.数据类型 数组名[数组长度] = {值1,值2,值3,...}；
    3.数据类型 数组名[] = {值1,值2,值3,...}；
## 访问方式：
    数组名[下标]； 下标从0开始；
- **利用for循环输出**
```
for (int i = 0; i < 5; i++){
    cout << arr[i] << endl;
}
```
