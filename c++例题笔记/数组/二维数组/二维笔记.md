## 二维数组的四种定义方式：
1. 数据类型 数组名[行数][列数];
2. 数据类型 数组名[行数][列数] = {{数据1，数据2},{数据3，数据4}};  
- 这种更直观，提高代码的可读性  
```
int arr[2][3] ={
    {1,2,3},
    {4,5,6}
}
```
> 
3. 数据类型 数组名[行数][列数] = {数据1，数据2，数据3，数据4};
4. 数据类型 数组名[][列数] = {数据1，数据2，数据3，数据4};

- 利用双for循环输出二维数组
```
for (int i = 0; i < 2; i++){
    for (int j = 0; j < 3; j++){
        cout<< arr[i][j] << endl;
    }
}
```
